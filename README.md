# Online Quiz App (Java Console Mini Project)

This is a simple **console-based quiz application** built using Java. It presents multiple-choice questions to the user, accepts answers, and calculates the final score.

---

## Objective

- Create a console-based quiz app.
- Demonstrate Java basics like classes, collections, input handling, and control flow.

---

## Features

- Multiple-choice questions
- Real-time score tracking
- Input validation for user responses

---

## Tools Used

- Java (JDK 8 or later)
- Console/Terminal for running the app

---


## How to Run

1. **Compile the files**:

```bash
javac Question.java QuizApp.java
```

2. **Run the app**:

```bash
java QuizApp
```
